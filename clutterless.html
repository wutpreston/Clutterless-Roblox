Clutterless Roblox Settings

Advertisements

Stop Showing Advertisements
Removes banner and sidebar ads from Roblox pages.
Stop Showing Sponsored
Removes the sponsored row from anywhere on roblox.

Home Page

Stop Showing Today's Picks
Remove the rows for Today's Picks.
Stop Showing Recommended For You
Removes the recommended for you rows.

Word Tweaks

Change "Charts" to "Games"
Restore the old name for experiences/charts.
Change "Communities" to "Groups"
Restore the old name for communities.
Change "Marketplace" to "Catalog"
Restore the old name for marketplace.

Games / Experiences

Blur Brainrot Games
Blurs games that include brainrot terminology in their titles.

Game Page Tweaks

Restore Green Play Button
Changes the blue play button back to the green button.

About

About Clutterless Roblox
Clutterless Roblox is an extension designed to remove all of the clutter roblox has been serving to the front page of users.

This extension also includes many QoL (Quality Of Life) features to restore roblox back to what it used to be before they went hardcore cooperate mode.
Subject To Breaking
At any point ROBLOX can update their website elements. This may cause unexpected results and/or functionalilty loss.

I will always try to fix these issues as they occure, but I cannot guarantee those updates to be quick.

Debug Mode
WARNING This may impact performance.

Turning this on will cause Clutterless Roblox to output everything it does into the Developer console.
This can be very memory intensive.

This should only be on if your trying to debug a problem.
